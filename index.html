✨ 증강

증강 정보

플레이어에게 추가적인 효과를 적용하는 시스템입니다.

실버

아이콘 증강명 설명
곡괭이 자동 수리 곡괭이의 내구도가 50% 이하일 때, 다이아몬드 채광 시 아주 낮은 확률로 내구도가 모두 수리됩니다.
다이아몬드를 낳는 거위 웅크리기를 2분 유지 시 마다 다이아몬드 1~3개를 랜덤으로 드랍합니다.
곡괭이 저격수 곡괭이의 사거리가 1.5배가 됩니다.
광전사 채굴꾼 몬스터 처치 후 30초 동안 채굴 속도가 증가합니다.
채굴 경험치 돌과 관련된 블럭 채굴 시 경험치로 즉시 변환됩니다.
존야갈걸퉷쒯 체력이 2 이하일 때 5초 동안 무적 상태가 됩니다. (쿨타임: 15분)
다이아 적금 인벤토리에 다이아 보유 시 1시간마다 이자를 획득합니다. (최대 1,000개 / 1%)

골드

아이콘 증강명 설명
광석 변환 광물을 채굴 시 일정 확률로 특정 범위 내에 있는 블럭이 랜덤한 광물 블럭으로 변환합니다.
도박꾼의 곡괭이 광물 채굴 시 일정 확률로 특정 인챈트의 레벨이 5~15 랜덤으로 설정됩니다.
압축 채굴 광물 채굴 시 일정 확률로 해당 광석을 블럭 형태로 획득합니다.
다이아 각성 다이아 64개를 소모하여 1분 동안 효율 혹은 행운 레벨이 +3 증가합니다. (쿨타임: 5분)
빵부기의 가호 빵부기의 응원으로, 30초간 효율과 행운 레벨이 3 증가하고, 이동속도가 10% 증가합니다. (쿨타임: 5분)
끊임없는 수리 걸어서 100블럭 이동 시 마다 곡괭이 내구도가 1 회복합니다.
채굴 콤보 채굴 시 스택이 증가하며, 스택 당 이동속도와 채굴속도가 증가합니다. 최대 스택 도달 시 15초 동안 행운 레벨이 증가합니다. (쿨타임: 15분 / 최대 스택: 100)

프리즘

아이콘 증강명 설명
거인화 몸집이 커지고, 채굴 속도가 낮아지지만 블럭을 2x3으로 채굴할 수 있습니다.
소인화 몸집이 작아지고, 채굴 속도가 증가하며, 이동속도가 빨라집니다. 대신 최대 체력이 50% 감소합니다.
행운의 곡괭이 행운 레벨이 증가합니다.
다이아가 보여 1분 동안 주변에 있는 광물을 투시합니다. (쿨타임: 5분)
굴착기 1분 동안 블럭을 3x3 으로 채굴할 수 있습니다. (쿨타임: 5분)
행운 2배로! 골드와 프리즘 증강의 등장 확률이 증가합니다, 추가로 행운과 효율의 레벨이 1 증가하고 이동속도가 10% 증가합니다.
자기장 폭발 광물 채굴 시 일정 확률로 범위 내 모든 광물을 파괴하고, 드랍한 광물을 자동으로 획득합니다.

시스템 개요

SOOP 방송 플랫폼의 후원을 통해 인게임에 직접 영향을 줄 수 있는 시스템입니다.

시청자들의 후원금에 따라 다양한 인게임 이벤트가 발생하며, 섭주 API를 통하여 접속한 스트리머 전체에게 미니게임 혹은 룰렛 2배 효과를 적용합니다!

후원 유형

유형 설명
💬 채팅 후원 일반적인 후원 유형입니다.
🎬 영상 후원 영상을 공유하는 후원 유형입니다.
🎯 미션 후원 미션을 부여하고, 미션 성공 시 인게임 이벤트가 발생하는 후원 유형입니다.
⚔️ 파티 후원 (대결 미션) 승리 여부에 관계없이 후원 시 후원한 방송에 이벤트가 발생하는 후원 유형입니다.

유저 API

후원 개수 효과 설명

섭주 API

후원 개수 효과 설명
444개 📝 영어/한글 받아쓰기 채팅과 화면에 표시되는 받아쓰기 게임을 진행
777개 ⬅️➡️ 방향키 리듬게임 화면에 표시되는 방향키를 보고 WASD를 이용해 맞추는 게임을 진행
999개 🎰 다이아 룰렛 2배 99초 목표 다이아몬드 룰렛 효과 2배 적용 (99초 지속)

작동 예시

스트리머 API 작동 예시

  • 시청자가 SOOP에서 1,111개 후원
  • → 서버에서 일반 상자 11개가 스트리머의 우편함으로 자동 지급됨
  • → 스트리머가 우편함을 확인하여 아이템 획득

섭주 API 작동 예시

  • 시청자가 444개 후원 → 영어/한글 받아쓰기 게임 진행

주의사항

  • 다이아 룰렛 2배 효과는 지속 시간이 중첩되어 여러번 후원 시 시간이 추가됩니다
